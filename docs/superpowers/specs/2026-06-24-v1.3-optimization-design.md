# Zen 消消乐 v1.3 — 成就感 + 玩法深入 + 音效 优化设计

> 日期：2026-06-24 | 状态：已确认

## 1. 成就感：localStorage 持久化 + 成绩面板

### 1.1 存储数据

| 字段 | key | 说明 |
|------|-----|------|
| 最高连消 | `zen_best_chain` | 单局最高连消波次 |
| 最高分 | `zen_best_score` | 单局最高得分 |
| 总消除数 | `zen_total_cleared` | 所有局累计消除宝石数 |
| 最高 Vibe | `zen_best_vibe` | 历史最高 Vibe 等级(1-5) |

### 1.2 成绩面板

- Zen HUD 旁加 🏆 按钮，点击弹出面板
- 面板内容：本次 vs 历史最佳 双列对比
- 每局结束自动更新 localStorage

## 2. 玩法深入：成就里程碑 + 道具强化

### 2.1 连消里程碑

| 连消数 | 称号 | 反馈 |
|--------|------|------|
| 5 | 初露锋芒 | 顶部横幅 |
| 10 | 渐入佳境 | 横幅 + 音效 |
| 15 | 炉火纯青 | 弹窗庆祝 |
| 20 | 天人合一 | 全屏粒子雨 |

- 首次达成弹窗，之后同级别用小横幅
- 成就数据存 localStorage

### 2.2 道具视觉强化

- 直线消除：触发位置增加光柱动画
- 炸弹消除：冲击波扩散
- 彩虹消除：全屏闪烁 + 粒子雨

## 3. 音效：Web Audio API 合成

| 音效 | 触发 | 实现 |
|------|------|------|
| 选中音 | 点击宝石 | 短促正弦波 440Hz |
| 交换音 | 交换动画 | 上升滑音 300→500Hz |
| 消除音 | 消除时 | 清脆三角波，匹配宝石颜色音调 |
| 连击音 | 连击升级 | 音调逐级升高(C4→C5) |
| 道具音 | 道具触发 | 和弦爆发 |
| 里程碑音 | 成就达成 | 上升琶音 |
